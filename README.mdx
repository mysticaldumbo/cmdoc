# cmdoc
view cmdoc/cptf documents easily with a simple command.
## Install
1. run the following commands:
```bash
git clone https://www.github.com/mysticaldumbo/cmdoc.git
cd cmdoc
g++ cmdoc.cpp -o cmdoc.exe
```
2. add the cmdoc folder to your path.
## Usage
to open command documents you can use the `cmdoc` command as in this example:
```bash
cmdoc example.cmdoc
cmdoc example.cptf
```
## More
if you want, you can also add the 'cds' folder to your path and use the 'cds' command. it's realy just there to show how easy it is to make cmdoc in batch compared to c++.
